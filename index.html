Architecture

Not a GPT wrapper.
A fully autonomous pipeline.

Krybit's three-stage engine hooks directly into your CI/CD. No chat boxes. No copy-paste. Just deterministic, agentic test generation that runs on every push.

01

Agentic Crawling

The agent spins up a headless browser and autonomously maps your entire UI surface — every route, every interactive element, every DOM mutation. It builds a living component graph that updates on every commit.

Headless Chromium DOM Diffing Component Graph
02

Logic Generation via Claude

Code diffs and the component graph are sent to Claude 3.5 Sonnet via the Anthropic API in agentic loops. Claude reasons about user flows, generates type-safe Playwright assertions, and self-heals broken selectors — all without human intervention.

Claude 3.5 Sonnet Anthropic API Agentic Loops Self-Healing
03

Native CI/CD Integration

Generated tests are committed directly to your repo and wired into GitHub Actions or GitLab CI pipelines. The agent opens PRs with test coverage diffs, runs the suite, and auto-merges on pass. Zero config.
